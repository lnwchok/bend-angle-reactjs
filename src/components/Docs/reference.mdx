## ทฤษฎีที่เกี่ยวข้อง ✨

การหาค่ามุมของ elbow ใช้วิธีการของ การคูณเวกเตอร์ด้วยเวกเตอร์แบบ Dot Product มาประยุกต์ใช้เพื่อหาคำตอบ และอาศัยการดัดแปลงเปลี่ยนจากทิศทางของท่อเป็นปริมาณเวกเตอร์ และที่ตำแหน่งของ elbow ให้เป็นจุดกำเนิดของเวกเตอร์ทั้งสอง แต่เนื่องจากค่ามุมของ elbow ที่ใช้กันนั้นคือมุมตรงข้ามกับมุม $\theta$ ที่คำนวณมาได้ ดังนั้นวิธีการคำนวณนี้จะต้องทำการเปลี่ยนเวกเตอร์ $\vec{a}$ ให้เป็นเวกเตอร์ตรงกันข้าม $(\vec{a}^{-1})$ ก่อนนำไปคำนวณโดยใช้สูตรข้างล่างนี้

### การคูณเวกเตอร์ด้วยเวกเตอร์แบบ Dot Product

การคูณเวกเตอร์แบบ dot product หรือ การหาปริมาณสเกลลาร์ (scalar product) ของเวกเตอร์ $\vec{a}$ และ $\vec{b}$ โดยที่ $\vec{a}=a_x+a_y+a_z$ และ $\vec{b}=b_x+b_y+b_z$ ตามลำดับ นั้นมีนิยามว่า ปริมาณสเกลลาร์หาได้จาก ปริมาณผลคูณของขนาดเวกเตอร์ $\vec{a}$, $\vec{b}$ และ $\cos$ ของมุมระหว่างเวกเตอร์ทั้งสอง และสามารถเขียนให้อยู่รูปสมการดังนี้

<div align="center">
  $\textbf{a}\cdot\textbf{b}=\lVert\vec{a}\rVert \lVert\vec{b}\rVert \cos\theta$
</div>

แต่สิ่งที่เราสนใจคือมุมระหว่าง $\vec{a}$ และ $\vec{b}$ ดังนั้นในการหาค่าของ $\theta$ สามารถจัดรูปสมการใหม่ได้ดังนี้

<div align="center">
$\theta=\cos^{-1}\left(\dfrac{\textbf{a}\cdot\textbf{b}}{\lVert\vec{a}\rVert \lVert\vec{b}\rVert}\right)$
</div>

โดยที่

- ขนาดเวกเตอร์ $\vec{a}$ หรือ $\lVert\vec{a}\rVert$ ซึ่งมีค่าเท่ากับ $\sqrt{{x^2} + {y^2} + {z^2}}$

- ปริมาณสเกลลาร์ของเวกเตอร์ $\vec{a}$ และ $\vec{b}$ หรือ $\textbf{a}\cdot\textbf{b}$ มีค่าเท่ากับ $a_x b_x + a_y b_y + a_z b_z$
